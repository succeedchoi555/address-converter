해외 직구할 때 가장 중요한 것은 정확한 주소 입력입니다. 배송 오류의 대부분이 잘못된 주소 입력 때문에 발생합니다.

해외 쇼핑몰 주소 입력란 이해하기

Address Line 1
상세 주소의 첫 번째 줄입니다. 아파트의 경우 동호수를 입력하세요.
예시: 101-dong 201-ho

Address Line 2
도로명 주소를 입력합니다.
예시: 152, Teheran-ro, Gangnam-gu

City
시 또는 구 이름을 입력합니다.
서울: Seoul
부산: Busan
성남시: Seongnam-si

State/Province
도 이름을 입력합니다. 서울은 비워두거나 Seoul을 입력하세요.
경기도: Gyeonggi-do
충청남도: Chungcheongnam-do

ZIP Code
5자리 우편번호를 입력합니다.
예시: 06236

자주하는 실수와 해결법

실수 1: 한글로 입력
잘못된 예: 서울시 강남구
올바른 예: Gangnam-gu, Seoul

실수 2: 순서가 뒤바뀜
잘못된 예: Seoul Gangnam-gu Teheran-ro 152
올바른 예: 152, Teheran-ro, Gangnam-gu, Seoul

실수 3: 우편번호 누락
우편번호는 배송의 핵심입니다. 반드시 입력하세요.

실수 4: 전화번호 형식 오류
국가번호 +82를 붙이고, 첫 자리 0을 제거하세요.
잘못된 예: 010-1234-5678
올바른 예: +82-10-1234-5678

쇼핑몰별 입력 가이드

아마존 (Amazon)
State 칸에 Seoul 또는 Gyeonggi-do를 입력하세요.
Phone number는 반드시 +82로 시작해야 합니다.
배송 옵션에서 한국 직배송 가능 여부를 확인하세요.

이베이 (eBay)
Country는 South Korea를 선택합니다.
Address Line 2는 선택사항이지만 입력을 권장합니다.
판매자에게 한국 배송 가능 여부를 문의하세요.

알리익스프레스 (AliExpress)
중국 배송이므로 특히 정확한 주소가 중요합니다.
한글 주소도 함께 입력할 수 있습니다.
배송 추적 번호를 꼭 저장하세요.

iHerb
건강식품 직구에 최적화된 사이트입니다.
한글 주소 입력도 지원합니다.
관세 걱정 없는 금액을 확인하세요.

배송 대행지 이용 시 주의사항

배송 대행 서비스를 이용한다면:

1. 대행지의 정확한 영문 주소를 확인하세요
2. 개인 고유번호(ID)를 정확히 입력하세요
3. 수령인 이름을 영문으로 작성하세요
4. 대행지 연락처를 정확히 입력하세요

배송비 절약 팁

무료 배송 금액을 확인하고 함께 주문하세요.
무게가 가벼운 제품을 선택하세요.
프로모션 기간을 활용하세요.

관세 피하는 방법

미국 직구: $200 이하
중국 직구: $150 이하 (일부 품목 제외)
유럽 직구: €150 이하

배송 추적하기

트래킹 번호를 저장하세요.
17track.net 같은 통합 추적 사이트를 이용하세요.
통관 단계에서 문자 알림을 확인하세요.

World Address로 실수 방지하기

저희 서비스를 사용하면 모든 쇼핑몰에 적합한 형식으로 주소가 변환됩니다. 복사-붙여넣기로 간편하게 입력하고, 철자 오류 걱정 없이 안전한 해외 직구를 즐기세요.
