해외 직구할 때 꼭 알아야 할 주소 입력 방법
해외 쇼핑몰 주소 입력란 이해하기

Address Line 1
상세 주소의 첫 번째 줄 (건물명, 동호수)
- 예: 101-dong 201-ho

Address Line 2  
상세 주소의 두 번째 줄 (도로명 주소)
- 예: 152, Teheran-ro

City
시/군 이름
- 서울: Seoul
- 부산: Busan

State/Province
도 이름
- 경기도: Gyeonggi-do
- 서울은 비워두거나 "Seoul"

ZIP Code
우편번호 (5자리)
- 예: 06236

자주하는 실수와 해결법

실수 1: 한글로 입력
❌ 서울시 강남구
✅ Gangnam-gu, Seoul

실수 2: 순서가 뒤바뀜
❌ Seoul Gangnam-gu Teheran-ro 152
✅ 152, Teheran-ro, Gangnam-gu, Seoul

실수 3: 우편번호 누락
배송 지연의 주요 원인! 반드시 입력하세요.

쇼핑몰별 입력 가이드

아마존 (Amazon)
- State 칸에 Seoul 또는 Gyeonggi-do 입력
- Phone number는 +82로 시작 (국가번호)

이베이 (eBay)
- Country는 South Korea 선택
- Address Line 2는 선택사항이지만 입력 권장

알리익스프레스 (AliExpress)
- 중국 배송이므로 특히 정확해야 함
- 한글 주소도 함께 입력 가능

배송 대행지 이용 시

해외 배송 대행 서비스를 이용한다면:
1. 대행지의 정확한 영문 주소 확인
2. 개인 고유번호(ID) 정확히 입력
3. 수령인 이름을 영문으로

World Address로 실수 방지하기

저희 서비스를 사용하면:
- 모든 쇼핑몰에 적합한 형식으로 변환
- 복사-붙여넣기로 간편 입력
- 철자 오류 제로

안전한 해외 직구, World Address와 함께하세요!
